# 拼豆模板生成器

一个基于React和TypeScript的拼豆（Perler Beads）模板生成工具。

## 🎯 项目特色

- 🖼️ 智能图片处理和上传
- 🎨 Delta E 2000高精度颜色匹配
- 📊 实时质量分析
- 📋 详细颜色清单
- 📄 多格式导出（PDF、PNG）

## 🚀 技术栈

- React 18 + TypeScript
- Vite + Tailwind CSS
- Canvas API 图像处理
- jsPDF PDF生成
- GitHub Pages 部署

## 📁 项目结构

完整的TypeScript + React项目，包含：
- 拼豆色卡数据管理
- 颜色匹配算法实现
- 图像处理工具
- PDF导出功能
- 响应式用户界面

🎊 开始使用：上传图片，调整设置，生成模板！
